# Forms Update

## Objetivo

Mantener formularios claros, validables y seguros.

## Reglas

- Labels explicitos
- Estados de error visibles
- Hint breve cuando agrega contexto real
- Validacion en cliente y servidor
- CTA principal claro

## Formularios sensibles

- Auth
- Mascotas
- Eventos
- Perfil del tutor
- Feedback

## Checklist rapido

- Campos obligatorios bien marcados
- Mensajes de error entendibles
- Disabled/loading correctos
- Mobile usable sin zoom forzado
