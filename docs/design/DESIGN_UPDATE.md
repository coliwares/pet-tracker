# Design Update

## Direccion visual

- Mobile-first
- Gradientes suaves
- Cards con bordes redondeados
- Jerarquia clara en titulares y chips
- Enfoque en legibilidad y acciones primarias

## Criterios

- Mantener consistencia entre dashboard, detalle y formularios
- Evitar ruido visual innecesario
- Dar feedback claro en estados vacios, errores y loading

## Componentes que marcan el tono

- `Header`
- `PetCard`
- `OnboardingPanel`
- `Button`
- `Input`

## Cuando tocar diseno

- Si cambia una jerarquia importante
- Si se agrega una accion nueva al dashboard
- Si una pantalla pierde claridad en mobile
